# Ticket M365-004 — Teams Login Issue

**User:** Tanya Brooks  
**Platform:** Microsoft 365 / Teams  
**Priority:** Medium  
**Category:** Teams / Authentication  
**Status:** Closed  

## Issue
User was unable to sign into Microsoft Teams. Login attempt failed despite valid credentials.

## Troubleshooting Steps
- Verified user account was active
- Checked sign-in status in Microsoft 365 Admin Center
- Verified Microsoft 365 Business Standard license was assigned
- Confirmed MFA was enabled
- Identified incomplete MFA registration during login process

## Resolution
Guided user through MFA registration process and reattempted sign-in. Teams access restored successfully.
