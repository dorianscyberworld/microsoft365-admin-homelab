# Microsoft 365 Admin Homelab

## Overview
This repository documents hands-on Microsoft 365 administration tasks completed in a Microsoft 365 Business Standard lab environment.

The purpose of this homelab is to practice real-world IT Help Desk and Microsoft 365 admin tasks including user management, password resets, MFA support, licensing, and access control.

---

## Tools Used
- Microsoft 365 Admin Center
- Microsoft 365 Business Standard
- Microsoft Entra ID
- Outlook
- Teams
- GitHub Documentation

---

## Skills Practiced
- User account management
- Password resets
- Multi-factor authentication support
- License assignment
- Access control
- Microsoft 365 troubleshooting
- Help Desk documentation
