# Procedure — Microsoft 365 Password Reset

## Purpose
Document the standard process for resetting a user password in Microsoft 365 Admin Center.

## Steps
1. Sign in to Microsoft 365 Admin Center.
2. Go to **Users → Active users**.
3. Search for the affected user.
4. Open the user profile.
5. Select **Reset password**.
6. Generate a temporary password.
7. Require the user to change password at next sign-in.
8. Provide temporary credentials securely.
9. Confirm the user can sign in.
10. Document the resolution.

## Security Notes
- Always verify user identity before resetting passwords.
- Do not send passwords through insecure channels.
- Require password change at next login.
- Document all account recovery actions.
